# JavaScript Practice Set — Summary

This repository contains 10 beginner-level JavaScript exercise solutions focused on strengthening core programming concepts such as functions, loops, conditionals, and basic algorithmic problem solving.

The tasks include:
- Checking even or odd numbers
- Evaluating student results (pass/fail)
- Finding the maximum of three numbers
- Calculating sum from 1 to N
- Generating multiplication tables
- Counting digits in a number
- Reversing numbers using math operations
- Computing factorial using loops
- Checking prime numbers
- Printing star patterns using nested loops

These exercises help build a strong foundation in logical thinking and JavaScript fundamentals.
